## 🛠️ COMMANDES RAPIDES

### Tester Localement
```bash
cd /home/user/webapp
pip install -r requirements.txt
python3 app.py
# Accès: http://localhost:5000
```

### Lancer les Tests
```bash
cd /home/user/webapp
python3 test_app.py --server
```

### Créer un Admin
```bash
cd /home/user/webapp
python3 create_admin.py
```

### Redémarrer le Serveur
```bash
pkill -f "python3 app.py"
cd /home/user/webapp && python3 app.py &
```
